1.O que é um repositório no Git e qual a sua importância?
R:é um armazenamento virtual para projetos. Ele permite que você salve versões do código, que você pode acessar quando precisar.

2.Qual a função do Git Bash?
R: Ele se torna um emulador para usar as linhas de comando do git

3.O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: Git add adiciona as mudanças feitas para o proximo commit. Git add <arquivo> adiciona apenas um arquivo em especifico, git add . adiciona todos os arquivos de uma vez.

4.Por que utilizamos o ponto (.) no comando git add .?
R: Para adicionar todos os arquivos alterados de uma vez em vez de um por um

5.Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R: Git commit cria um commit dos seus arquivos e os preparam para serem enviados para o repositorio. É importante por uma mensagem para saber oque foi feito em cada commit

6.Qual a função do comando git push?
R: Git push serve para enviar os commits feitos para o repositorio criado na pasta do arquivo em que esta sendo usado

7.Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R: Para saber cada alteração feita em cada commit

8.Explique a função das tags HTML e como elas estruturam uma página.
R: As tags main, header e footer são elementos que definem a estrutura e o conteúdo de uma página web

9.O que são atributos em uma tag HTML e qual a sua função?
R: são informações adicionais que controlam o comportamento de um elemento HTML

10.O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
R: O emmet é uma ferramenta que ajuda a escrever HTML, CSS e outras linguagens de forma mais eficiente e rapida. Aumenta sua eficiancia utilizando o html e sera mais rapido.

11.O que é o box-sizing no CSS e por que usamos ele no reset inicial?
R: Ele define a largura e altura dos elementos não calculados. Ele pode ser usado no reset inicial para definir o tamanho maximo da pagina.

12.Por que é importante utilizar tags semânticas no HTML?
R: Porque elas melhoram a acessibilidade

13.O que são media queries no CSS e qual a sua função?
R: São um recurso do CSS  que permite adaptar a visualização de uma pagina a diferentes dimensões de tela.

14.Qual a função do GitHub Pages?
R: Cria uma pagina do repopsitorio git adicionado no github

15.O que é o Flexbox no CSS e para que ele é utilizado?
R: É uma ferramenta do CSS que permite criar layouts flexíveis, distribuindo espaços entre elementos de forma eficiente

16.Qual a diferença entre justify-content e align-items em um container Flexbox?
R: A diferença entre justify-content e align-items é o eixo em que os itens são alinhados

17.Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
R: A propriedade flex-grow especifica o quanto o item crescerá em relação ao restante dos itens flexíveis dentro do mesmo contêiner.

18.Explique a diferença entre flex-direction: column; e flex-direction: row;.
R: A diferença entre flex-direction: column; e flex-direction: row; é que o primeiro organiza os itens em colunas, enquanto o segundo organiza os itens em linhas.
